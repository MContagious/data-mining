  • Lists all the application-id's found in the ULGFile
  • The list can be filtered by program-id name or date it was called as listed in ULGFile
  • Each program-id link/Button lists
    • The databases it is using
    • Variables and their values fetched from database(tables)
  • Lists all the databases found in the revision control
  • By default all the database details were hidden in the web page as the list is huge and impacting the performance
  • Click on the header of the database to view details
  • Clicking on the header again will hide the information
  • Shows the program-id dependency graph
  • Node Colors and their meaning
    • steelblue : Collapsed node
    • lightblue : Node has children
    • lightcoral : The program was not found/not source controlled in revision control system
    • lightgreen : The program has no dependencies(End node)
    • yellow : The node indicates a variable.
